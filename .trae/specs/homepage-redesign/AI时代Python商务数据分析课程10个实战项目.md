## 项目 1 零售门店经营数据 Python 全流程清洗与 AI 辅助基础洞察

### 项目定位

课程入门项目，巩固 Python 数据分析核心基础，完成从 Excel/SQL 到 Python 的能力迁移，入门 AI 商务分析应用

### 🔴 前置必学知识模块（先学内容）

1. Python 核心库进阶：pandas 深度应用（数据读取、缺失值 / 异常值处理、数据类型转换、分组聚合、透视表、多表拼接）、numpy 基础数值计算

2. Python 商务可视化基础：matplotlib、seaborn 基础图表绘制（柱状图、折线图、饼图、箱线图），适配商务汇报场景

3. 企业级数据规范：商务数据清洗标准、异常值识别逻辑、数据口径统一方法

4. AI 入门基础：大模型 API 基础调用方法、商务分析场景的 Prompt 工程基础

### 项目核心业务目标

针对线下连锁零售门店的月度经营数据，完成全流程数据清洗与标准化，拆解营收、客单价、坪效、毛利率等核心经营指标，完成门店经营排名、品类销售结构分析，定位经营异常门店，最终通过 AI 生成基础洞察与可落地的优化方向

### AI 时代核心设计

告别传统 “只做数据统计 \+ 图表绘制” 的基础分析，通过大模型 API 自动对异常经营数据做根因初步诊断，自动生成标准化的经营分析摘要，替代传统手动撰写分析结论的模式，实现分析效率的指数级提升

### 核心技术栈

Python（pandas/numpy/matplotlib/seaborn）、Excel（原始数据导入与校验）、大模型 API（豆包 / 通义千问）

### 项目最终交付

数据清洗全流程代码、经营指标分析代码、可视化图表、AI 生成的门店经营洞察报告、课程作业报告

### 项目核心业务目标

Python（pandas/prophet/statsmodels/matplotlib）、Excel（库存数据校验）、PowerBI（销量预测看板）、大模型 API

### 项目最终交付

时序预测全流程代码、安全库存计算模型、AI 库存预警与补货建议生成工具、销售预测与库存监控看板、项目分析报告
